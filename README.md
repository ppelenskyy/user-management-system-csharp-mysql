# Sistema de Gestión de Usuarios - C#

Aplicación desarrollada en C# como proyecto académico del ciclo DAM (Desarrollo de Aplicaciones Multiplataforma).

## Descripción

Sistema de gestión de usuarios que permite:

- Dar de alta usuarios
- Eliminar usuarios mediante DNI
- Modificar un usuario
- Buscar usuarios por DNI
- Mostrar todos los usuarios registrados
- Validar datos de entrada
- Gestionar errores de acceso a base de datos

Aplicación desarrollada en consola aplicando Programación Orientada a Objetos y acceso a bases de datos MySQL.

## Tecnologías utilizadas

- C#
- .NET
- MySQL
- MySql.Data
- Programación Orientada a Objetos
- Expresiones Regulares (Regex)
- Consultas SQL parametrizadas
- Manejo de excepciones

## Estructura

`Program.cs` → Punto de entrada
`GestionBD.cs` → Gestión de operaciones con la base de datos
`Utils.cs` → Validación y lectura de datos
Base de datos MySQL → Almacenamiento de usuarios
